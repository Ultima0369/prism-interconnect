## 🛠️ 制作工具建议

### 免费工具
1. **Canva**：模板设计、社交媒体图片
2. **Figma**：专业设计、协作设计
3. **DaVinci Resolve**：视频编辑（免费版功能强大）
4. **Blender**：3D动画、粒子效果
5. **Inkscape**：矢量图形设计

### 简单生成脚本
```python
# 简单的方程图片生成脚本（使用Pillow）
from PIL import Image, ImageDraw, ImageFont

# 创建图片
img = Image.new('RGB', (1200, 630), color='#0a192f')
draw = ImageDraw.Draw(img)

# 添加方程
font_large = ImageFont.truetype("arial.ttf", 80)
font_small = ImageFont.truetype("arial.ttf", 30)

draw.text((300, 250), "E=mc²", fill="#64ffda", font=font_large)
draw.text((700, 250), "1+1>2", fill="#ff79c6", font=font_large)
draw.text((500, 350), "|", fill="#ffffff", font=font_large)

# 添加文字
draw.text((400, 450), "两个方程，一张底牌", fill="#ffffff", font=font_small)

img.save("two-equations.png")
```
