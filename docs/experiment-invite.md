# 🔥 棱镜之夜：第一次火堆旁对话邀请

## 🎯 邀请对象

**你，正在读这段话的人。**

如果你：
- 愿意尝试新的对话方式
- 对自我认知和深度思考感兴趣
- 有15分钟的时间和一个真实困惑
- 愿意分享简单的体验反馈

**那么，火堆旁有你的位置。**

## 🧠 我们要做什么？

### 一次15分钟的认知实验

1. **选择一个真实困惑**（工作、关系、自我、未来...）
2. **使用棱镜Web应用**体验多元视角对话
3. **在留白中感受思考过程**
4. **分享你的体验和反馈**

### 这不是"解决问题"，而是"看见问题"

我们不做咨询，不给答案。
我们只是提供一个镜子，让你看见自己的思考光谱。

## 🌈 体验流程

### 第一步：访问棱镜体验版
**链接**: [点击这里体验](https://github.com/Ultima0369/prism-interconnect#极简棱镜三秒呼吸体验)

或者本地运行：
```bash
# 克隆项目
git clone https://github.com/Ultima0369/prism-interconnect.git
cd prism-interconnect

# 运行极简体验版
streamlit run examples/streamlit_simple_app.py
```

### 第二步：写下你的困惑
**建议主题**（选择一个）：
- 🏢 **工作困惑**：职业方向、工作压力、团队关系
- 💖 **关系困惑**：亲密关系、家庭沟通、朋友相处
- 🧠 **自我困惑**：拖延、焦虑、自我价值、人生意义
- 🌱 **成长困惑**：学习困难、技能提升、习惯养成
- 🔮 **未来困惑**：人生规划、重大决策、不确定性

**示例**：
> "我最近总是感到焦虑，明明工作顺利，家庭和睦，但就是有一种说不出的不安。"

### 第三步：体验三个光谱
**点击三个按钮，体验不同视角**：

1. 🔥 **红色光谱 · 直觉**
   - 你的本能感受是什么？
   - 身体有什么反应？
   - 第一直觉想做什么？

2. 🔵 **蓝色光谱 · 逻辑**
   - 事实和数据是什么？
   - 逻辑推理的结论？
   - 可能的原因和结果？

3. 🟣 **紫色光谱 · 反思**
   - 你如何看待自己的思考？
   - 这个困惑背后的模式？
   - 从中学到了什么？

### 第四步：在留白中停顿
**不要急着找答案，先感受：**
- 哪个视角最触动你？
- 留白时你在想什么？
- 有什么新的感受浮现？
