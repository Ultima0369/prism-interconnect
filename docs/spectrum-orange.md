# 橙色光谱 (Orange Spectrum)

## 概述

橙色是可见光谱中介于红色和黄色之间的颜色，波长范围约为 **590-620 纳米**。它是暖色调的代表，融合了红色的热情与黄色的明亮，象征着活力、创造力和温暖。

## 物理特性

### 波长范围
- **起始波长**: 590 nm
- **结束波长**: 620 nm
- **中心波长**: 约 605 nm

### 光学特性
- **频率**: 约 480-510 THz
- **光子能量**: 约 2.00-2.10 eV
- **色温**: 暖色调，约 2000-3500K

## 实用建议

### 使用技巧
1. **平衡使用** - 橙色作为强调色效果最佳
2. **考虑环境** - 自然光下更温暖，人工光下可能偏黄
3. **受众考量** - 不同文化对橙色的接受度不同
4. **可访问性** - 确保与背景有足够对比度

### 避免的情况
- 医疗环境（可能引起焦虑）
- 正式商务场合（过于随意）
- 大面积卧室使用（可能影响睡眠）
