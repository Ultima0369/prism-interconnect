## 🔧 故障排除

### 常见问题

#### 安装问题
**Q: 安装依赖失败**
```bash
# 解决方案
pip install --upgrade pip
pip install -r requirements.txt --no-cache-dir
```

**Q: 导入模块失败**
```python
# 确保在正确目录
import sys
sys.path.append('/path/to/prism-interconnect')
```

#### 运行问题
**Q: Streamlit应用无法启动**
```bash
# 检查端口占用
netstat -ano | findstr :8501  # Windows
lsof -i :8501                 # macOS/Linux

# 使用其他端口
streamlit run app.py --server.port 8502
```

**Q: 性能问题**
```python
# 调整性能参数
from prism_sdk import PrismClient
client = PrismClient(
    max_workers=4,      # 调整并发数
    timeout=30,         # 调整超时时间
    cache_enabled=True  # 启用缓存
)
```

#### 配置问题
**Q: 环境变量不生效**
```bash
# 检查环境变量
echo $PRISM_ENV  # Linux/macOS
echo %PRISM_ENV% # Windows

# 设置环境变量
export PRISM_ENV=production  # Linux/macOS
set PRISM_ENV=production     # Windows
```
