# 🔮 棱镜互联协议 (Prism Interconnect Protocol)

> *"在比特之上构建意义，在信息之上构建理解"*  
> *—— 意义层互联网宣言*

## 🌟 项目概述

**棱镜互联协议**是一个开创性的开源项目，旨在定义和实现**意义层互联网**——在传统互联网的信息传输层之上，增加一个专门用于传输和理解意义的协议层。

### 🎯 核心愿景

我们不是在建造"更好的聊天机器人"，而是在建造**认知伙伴**。我们不是在优化"信息传输"，而是在定义**意义交换**。

## 🔥 核心特性

### 1. **多元视角强制**
```python
# 每个响应必须包含至少三种视角
response = {
    'red_spectrum': '直觉和身体感受',
    'blue_spectrum': '分析和逻辑推理', 
    'purple_spectrum': '元认知和反思'
}
```

### 2. **留白必需**
```python
# 对话中必须包含留白
whitespace = {
    'duration': 'variable',  # 基于认知复杂度
    'purpose': '神经整合和意义生成',
    'enforced': True
}
```

### 3. **递归深度限制**
```python
# 防止无限递归
recursion = {
    'max_depth': 3,

    'auto_stop': True
}
```

### 4. **非评判性原则**
```python
# 禁止比较视角优劣
ethics = {
    'non_judgmental': True,
    'perspective_equality': True,
    'no_right_wrong': True
}
```

## 🚀 快速开始

### 1. **安装**
```bash
# 克隆仓库
git clone https://github.com/Ultima0369/prism-interconnect.git
cd prism-interconnect

# 安装依赖
pip install -r requirements.txt

# 运行示例
python examples/basic_prism_dialogue.py
```

### 2. **基本使用**
```python
from prism.protocol import PrismProtocol

# 初始化棱镜协议
prism = PrismProtocol()

# 开始棱镜对话
response = await prism.dialogue(
    query="什么是真正的理解？",
    context={
        'user_background': '哲学爱好者',
        'depth_preference': 'deep'
    }
)

# 输出多元响应
print("🔴 红色光谱（直觉）:", response.red_spectrum)
print("🔵 蓝色光谱（分析）:", response.blue_spectrum)  
print("🟣 紫色光谱（元认知）:", response.purple_spectrum)
print("⏸️  建议留白:", response.whitespace_suggestion)
```

### 3. **高级配置**
```python
# 自定义棱镜配置
config = {
    'spectra': {
        'red': {'weight': 0.3, 'style': 'intuitive'},
        'blue': {'weight': 0.4, 'style': 'analytical'},
        'purple': {'weight': 0.3, 'style': 'metacognitive'}
    },
    'whitespace': {
        'min_duration_ms': 1000,
        'max_duration_ms': 10000,
        'adaptive': True
    },
    'ethics': {
        'enforce_non_judgmental': True,
        'require_transparency': True,
        'safety_checks': 'strict'
    }
}

prism = PrismProtocol(config=config)
```
