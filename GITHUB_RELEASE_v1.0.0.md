## 🎯 立即开始

### 1. **安装 Python SDK**
```bash
# 从 GitHub 安装
pip install git+https://github.com/Ultima0369/prism-interconnect.git@v1.0.0

# 或克隆仓库
git clone https://github.com/Ultima0369/prism-interconnect.git
cd prism-interconnect
git checkout v1.0.0
cd implementations/python
pip install -e .
```

### 2. **使用棱镜对话**
```python
from prism_sdk import PrismClient

client = PrismClient(api_key="YOUR_API_KEY")
response = client.prismatic_dialogue(
    query="如何平衡工作与生活？",
    max_depth=3,
    require_whitespace=True
)

print(f"红色光谱: {response.spectra.red}")
print(f"蓝色光谱: {response.spectra.blue}")
print(f"紫色光谱: {response.spectra.purple}")
print(f"留白洞见: {response.whitespace.insights}")
```

### 3. **安装 OpenClaw Skill**
```bash
# 一键安装
openclaw skill install https://github.com/Ultima0369/prism-interconnect/implementations/openclaw/skill.yaml

# 基本使用
@棱镜 如何提高我的决策质量？
棱镜分析一下当前AI发展的伦理困境
留白反思我刚才的决策过程
多视角看待气候变化问题
```
